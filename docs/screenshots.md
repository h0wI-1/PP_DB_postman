# Shot list скриншотов

Общие требования ко всем скриншотам:
- только **светлая** тема (белый фон), тёмная тема не допускается;
- в Postman включить светлую тему: Settings → Themes → Light;
- окна по возможности на полный экран, интерфейс чистый;
- подписи к рисункам в едином стиле: `Рисунок N – <пояснение>`;
- выделять важное красной рамкой или стрелкой (одним цветом во всех кадрах).

## Как быстро переключить Postman в светлую тему
`Postman → Settings (⌘ ,) → Themes → Light`. После этого все окна Postman
становятся белыми, что соответствует требованиям к оформлению скриншотов.
